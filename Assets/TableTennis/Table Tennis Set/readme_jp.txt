このアセットはUnity 2017.1.0f3によって作成されています。

このアセットには卓球の試合を再現できるようにするための7つのオブジェクト、及びそのメッシュ・シェーダー・テクスチャなどが含まれます。
3DモデルのおおよそのColor, Roughnessなどをインスペクタから調整できます。
スコアボードのスコアは、インスペクタのマテリアルのScoreの数値を変更することで表現できます。
各モデルに含まれるUCXプレフィックスの付いたメッシュは当たり判定用に用意されたものです。

■概要
- メッシュ : 13（UCX用のモデルを除く）
- マテリアル : 23
- テクスチャ : 69
- シェーダー : 14

■使用方法
"Table Tennis Set"フォルダ直下の"Prefabs"にあるプレハブを追加することで使用できます。
3Dモデルの階層構造や、モデルを見たい場合は"Scene"の"overview"がサンプルシーンとなっていますので、そちらをご覧ください。
